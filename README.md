# Mouse4 - Windows 空白处双击返回上一级 🖱️

Mouse4 是一个轻量级的 Windows 效率工具。它允许你在文件资源管理器（Explorer）的 **空白处双击鼠标左键**，直接返回上一级目录。

从此告别在这个巨大的屏幕上寻找微小的“向上”箭头！🚀

## ✨ 主要功能

* **双击空白处返回**：在文件夹空白区域双击，自动模拟 `Alt + Up` 返回上一级。
* **智能防误触**：自动识别并忽略文件、图片、文本框等控件，只响应真正的“空白背景”。
* **系统托盘常驻**：程序启动后自动最小化到右下角托盘，不占用任务栏空间。
* **极简静默**：无主窗口，无打扰，后台低功耗运行。

## 📥 下载与安装

1.  访问本项目的 [Releases 页面](../../releases)。
2.  下载最新的 `mouse4.exe` 文件。
3.  直接双击运行即可。

> **💡 小技巧：如何开机自启？**
> 1. 按下 `Win + R` 键。
> 2. 输入 `shell:startup` 并回车。
> 3. 将 `mouse4.exe` 的快捷方式拖入打开的文件夹中即可。

## 🛠️ 使用说明

* **运行状态**：双击程序后，它会隐藏在电脑右下角的系统托盘（时间旁边的小箭头里）。
* **功能菜单**：右键点击托盘中的图标，可以：
    * 访问 GitHub 主页。
    * **Exit** (退出程序)。
* **注意**：如果是第一次运行，Windows Defender 可能会提示拦截，点击“更多信息” -> “仍要运行”即可。
